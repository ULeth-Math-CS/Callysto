We can certainly get started rewriting \(4\log_{2}(x) + 3\) by applying the Power Rule to \(4\log_{2}(x)\) to obtain \(\log_{2}\left(x^4\right)\text{,}\) but in order to use the Product Rule to handle the addition, we need to rewrite \(3\) as a logarithm base \(2\text{.}\) From TheoremÂ 1.1, we know \(3 = \log_{2}\left(2^3\right)\text{,}\) so we get

\begin{align*} 4\log_{2}(x) + 3 \amp = \log_{2}\left(x^4\right) + 3 \tag{Power Rule}\\ \amp = \log_{2}\left(x^4\right) + \log_{2}\left(2^3\right) \tag{Since $3 = \log_{2}\left(2^3\right)$}\\ \amp = \log_{2}\left(x^4\right) + \log_{2}(8)\\ \amp = \log_{2}\left( 8x^4\right) \tag{Product Rule} \end{align*}
