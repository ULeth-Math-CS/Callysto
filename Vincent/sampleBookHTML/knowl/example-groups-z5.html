Example3.2.2

The integers mod \(n\) form a group under addition modulo \(n\text{.}\) Consider \({\mathbb Z}_5\text{,}\) consisting of the equivalence classes of the integers 0, 1, 2, 3, and 4. We define the group operation on \({\mathbb Z}_5\) by modular addition. We write the binary operation on the group additively; that is, we write \(m + n\text{.}\) The element 0 is the identity of the group and each element in \({\mathbb Z}_5\) has an inverse. For instance, \(2 + 3 = 3 + 2 = 0\text{.}\) Figure Figure 3.2.3 is a Cayley table for \({\mathbb Z}_5\) (Contributed by Robert Beezer). By Proposition Proposition 3.1.4, \({\mathbb Z}_n = \{0, 1, \ldots, n-1 \}\) is a group under the binary operation of addition mod \(n\text{.}\)
